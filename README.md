# OPAL-OKB — САПР оптики

> Портирование расчётного ядра **OPAL-PC** (MS-DOS, 1990-е) на **Python 3.10+ / PyQt5**

Расчёт, анализ, оптимизация и визуализация оптических систем: линз, объективов, окуляров, зеркально-линзовых систем.

## Установка и запуск

```bash
pip install PyQt5 numpy
py main.py
```

- **Python 3.10+** (рекомендуется 3.13)
- На Windows используется `py` launcher

## Возможности

### Расчёт
- **Параксиальный расчёт** — y-nu метод: f', BFD, FFD, кардинальные точки
- **Суммы Зейделя** SI–SV: сферическая, кома, астигматизм, кривизна поля, дисторсия
- **Реальная трассировка лучей** — векторный закон Снеллиуса, TIR, OPL
- **Асферические поверхности** — конические (k) + полиномиальные A4–A10
- **Многоволновый расчёт** — до 5 длин волн одновременно

### Каталог стёкол
- **891 марка**: ГОСТ (14 базовых + КВАРЦ/КВАРЦСТК/ОФ1/ТК20) + Schott + HOYA + Ohara
- Формула Герцбергера для n(λ)
- **n_override** — точные показатели преломления из LBO-файлов OPAL-PC

### Библиотека оптических систем
- **612 систем** в 13 библиотеках (.LBO):
  - LENS (116) — объективы (Индустар, Юпитер, Мир)
  - OCULAR (76) — окуляры
  - RUSSAR (84) — широкоугольные объективы
  - BINOCUL (22) — бинокли
  - и др.
- **Декодер LBO** — полное reverse engineering бинарного формата:
  - Все поля заголовка расшифрованы (см. `docs/LBO_FORMAT.md`)
  - Поддержка зеркал (glass_index=65535)
  - Glass names: smart split (КВАРЦСТК, склеенные имена)
  - Тип предмета/изображения, апертура (NA/Y/F#), диафрагма (ND+SD)
  - Стандартные длины волн e, G', C по умолчанию

### Поддержка зеркал
- Зеркально-линзовые системы (катадиоптрические)
- Визуализация: золотистый контур + штриховка отражающей стороны
- `paraxial_trace()`: n sign flip + reflection formula
- `seidel_aberrations()`: mirror-aware

### Параксиальные характеристики
- Мульти-λ таблица: f', sF', Δf' для каждой длины волны
- Кардинальные отрезки: sF, sF', sH, sH', sP, sP'
- Г.ММСС формат угловых полей

### Формирование оптической системы
- Тип предмета/изображения: дальнего/ближнего типа
- Передний отрезок, смещение от пл. Гаусса (мм/дптр)
- Диафрагма: ND (поверхность) + SD (смещение)
- Передняя/задняя апертура: Y/NA/F/#

### Визуализация
- Ход лучей в стиле OPAL-PC (чёрный фон, зелёные лучи, голубые линзы)
- **Хроматические лучи** — кнопка «Хром.Ц»: все длины волн разными цветами
- Зум (колесо мыши), пан (drag), крестокурс с координатами
- Полноэкранный режим (двойной клик)
- Таблица лучей

### Оптимизация
- **DLS** (Damped Least Squares)
- **Nelder-Mead Simplex**

### Подгонка характеристик
- f', BFD, увеличение — binary search по радиусу или толщине

### Ахроматизация
- Автоматический расчёт R для пар стёкол (крон + флинт)

### Импорт / Экспорт
- **.OPJ** — файлы OPAL-PC (standalone + из LBO-библиотек)
- **.opal.json** — человекочитаемый JSON

### Системные операции
- Оборачивание (reverse)
- Масштабирование
- Стандартные радиусы ГОСТ 1807-75 (ряд R40)

### Многопоточный расчёт
- **Фаза 1** (мгновенно): параксиальный расчёт + ход лучей
- **Фаза 2** (фон): все 23 вкладки анализа параллельно (до 8 потоков)
- Индикатор «⏳ Расчёт...» во время вычислений

| `main.py` | Главное окно GUI |

| `advanced_analysis.py` | PSF, LSF, ENC, PTF |
| `zernike.py` | Полиномы Цернике + хроматика |
