# OPAL-OKB — САПР оптики

> Портирование расчётного ядра **OPAL-PC** (MS-DOS, 1990-е) на **Python 3.10+ / PyQt5**

Расчёт, анализ, оптимизация и визуализация оптических систем: линз, объективов, окуляров, зеркально-линзовых систем.

## Установка и запуск

```bash
pip install PyQt5 numpy
py main.py
```

- **Python 3.10+** (рекомендуется 3.13)
- На Windows используется `py` launcher (`python` тоже работает)

## Возможности

### Расчёт
- **Параксиальный расчёт** — y-nu метод: f', BFD, FFD, кардинальные точки
- **Суммы Зейделя** SI–SV (исправленные, ненулевые): сферическая, кома, астигматизм, кривизна поля, дисторсия
- **Реальная трассировка лучей** — векторный закон Снеллиуса, TIR, OPL (оптическая длина пути)
- **Асферические поверхности** — конические (k) + полиномиальные A4–A10

### Каталог стёкол
- **889 марок**: ГОСТ + Schott + HOYA + Ohara
- Формула Герцбергера: `n(λ) = C₀ + C₁λ² + C₂λ⁴ + C₃L + C₄L² + C₅L³`

### Анализ (13 вкладок)
| # | Вкладка | Описание |
|---|---------|----------|
| 1 | Точечная диаграмма | Пятно рассеяния, RMS |
| 2 | Δy' | Поперечная аберрация |
| 3 | Δs' | Продольная аберрация |
| 4 | W (OPL) | Волновая аберрация |
| 5 | ЧКХ геометрическая | Геом. MTF через FFT |
| 6 | ЧКХ дифракционная | Дифр. MTF через FFT |
| 7 | Дисторсия | По полю |
| 8 | Астигматизм + кома | По полю |
| 9 | Фокусировка | RMS vs defocus |
| 10 | PSF | Point Spread Function |
| 11 | LSF | Line Spread Function |
| 12 | ENC | Encircled Energy |
| 13 | PTF | Phase Transfer Function |

### Полихроматический анализ
- Взвешенный по λ расчёт для всех длин волн одновременно

### Визуализация
- Ход лучей в стиле OPAL-PC (чёрный фон, зелёные/красные лучи, голубые линзы)
- Зум (колесико), пан (drag), полноэкранный режим (двойной клик)

### Оптимизация
- **DLS** (Damped Least Squares)
- **Nelder-Mead Simplex**

### Подгонка характеристик
- f', BFD, увеличение — binary search по радиусу

### Ахроматизация
- Автоматический расчёт R для 3 пар стёкол (крон + флинт)

### Импорт / Экспорт
- **.OPJ** — 83 файла из архива OPAL-PC (63 с именами стёкол)
- **.opal.json** — человекочитаемый JSON

### Системные операции
- Оборачивание (reverse)
- Масштабирование
- Стандартные радиусы ГОСТ 1807-75 (ряд R40)

### Прочее
- Множественные точки поля (до 5) + виньетирование
- Положение диафрагмы (stop surface)
- Диаграмма стёкол (nd–νd)
