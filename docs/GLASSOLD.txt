 СИСТЕМА АВТОМАТИЗИРОВАННОГО ПРОЕКТИРОВАНИЯ ОПТИЧЕСКИХ СИСТЕМ
                       OPAL - PC

             РУКОВОДСТВО ДЛЯ ПОЛЬЗОВАТЕЛЕЙ

             Каталоги оптических материалов

    Для удобства   задания   значений   показателей  преломления
оптической  среды  используется  машинный   каталог   оптических
материалов.

    С OPAL-PC поставляются следующие машинные каталоги:
     - российский каталог оптических материалов, включающий стекла
       из ГОСТа 13658-78, стекла серий 100, 200, 500, жидкости,
       кристаллы, пластмассы, ИК-материалы и пр.
     - каталог стекла фирмы SHOTT    (Германия);
     - каталог стекла фирмы SOVIRELL (Франция);
     - каталог стекла фирмы OHARA    (Япония);
     - каталог стекла фирмы HOYA     (Япония).

             1. Представление показателей преломления.

     В каталогах  оптических  материалов  OPAL-PC  для  описания
показателей   преломления   оптических  сред  n(Л)  используются
аппроксимирующие формулы:

     - для  сред  в  широком  диапазоне  волн,  а  так   же   ИК
       материалов,     кристаллов     и     т.д.    используется
       модифицированная формула Герцбергера:

       n(Л) = C0 + C1*Л^2 + С2*Л^4 + C3*L + C4*L^2 + C5*L^3 + ...

     где: L=1/(Л^2 - Л0^2),  Л0 - выбирается  в  зависимости  от
          рабочего интервала длин волн, Л - длина волны в мкм.

     - для  оптических  стекол,  особенно  каталогов иностранных
       фирм,  рабочий интервал длин волн которых  0.365  -  1.04
       мкм,   используется   формула   фирмы   SHOTT   с  шестью
       коэффициентами:

         n(Л)^2 = a0 +a1*Л^2 +a2*Л^-2 +a3*Л^-4 +a4*Л^-6 +a5*Л^-8

     Используемые формулы  обеспечивают   получение   показателя
преломления  в  рабочем  интервале  длин  волн с точностью  5-10
единиц шестого знака.

             2. Информация, хранящаяся в каталогах.

     В машинных каталогах OPAL-PC для каждого материала хранится
следующая информация:
     - марка  оптического  материала  и  его  код   по   данному
       каталогу.  Марка  содержит не более 8 символов ,код имеет
       четыре цифры - первые  две  определяют  номер  группы,  к
       которому принадлежит этот материал, например, крон,флинт,
       кристалл.  Следующие  две  цифры  это  порядковый   номер
       материала  в  группе.  Ссылка на материал возможна как по
       марке,  так и по коду ,  при этом заглавные  и  прописные
       буквы не различаются;
     - рабочий диапазон длин волн в мкм ;
     - коэффициенты интерполяционной формулы;
     - числа Аббе для ND и Ne.
     - плотность;
     - температурные изменения показателя преломления;
     - коэффициент линейного расширения для нескольких интервалов
       температур;
     - поглощение;

            3. Использование машинных каталогов.

     Каталоги оптических  стекол  используются  при формировании
оптической системы.  По указанной пользователем марке или коду и
требуемым  длинам  волн  производится  восстановление показателя
преломления.  Если требуемая  длина  волны  лежит  вне  рабочего
интервала этого материала, выдается сообщение об ошибке.
     Данные о плотности оптического материала  используются  при
определении   массы  линз.  Термооптические  и  теплотехнические
характеристики - при  определении  термоустойчивости  оптической
системы.
     Для стекол   российского   каталога    возможен    пересчет
показателей  преломления по паспортным данным плавки оптического
стекла.

         4. Просмотр характеристик оптических материалов,
                поиск аналогов, диаграмма стекол.

     Просмотр характеристик оптических материалов,  занесенных в
машинный каталог,  производится специальной  работой  -  КАТАЛОГ
СТЕКОЛ.  В  данной  работе  для  любого  материала  из машинного
каталога можно  просмотреть  показатель  преломления  для  любой
длины волны из рабочего интервала этого материала, а также любую
характеристику, описанную в п.2.

         4.1. Управление работой.
     На экране постоянно находится меню работы ,  окно настройки
параметров, строка с назначениями управляющих клавиш. Переход из
окна  меню  в  окно  настройки  и  обратно производится клавишей
"TAB". Смена каталога материалов - клавиша "F2".
    Выполнение пункта   меню   осуществляется  нажатием  клавиши
"ENTER".  Вывод производится либо в окно  находящееся  в  нижней
части  экрана  либо  на  полный  экран.  Изменение  окна  вывода
производится клавишей "F5".  Если вся информация не помещаются в
окно  вывода ,то строки можно передвигать вверх и вниз клавишами
движения курсора.  Возврат из окна просмотра в меню производится
клавишей  "ESC".  Выведенные характеристики можно распечатать на
принтере ( клавиша "F7" ).  Если принтер находится  в  нерабочем
состоянии  или отсутствует ,то предлагается вывести информацию в
файл с именем GLASS.PRN .

        4.2. Задание материалов.
     Первые два    пункта   меню   предназначены   для   задания
материалов,  для  которых  выводятся  требуемые  характеристики.
Характеристики  выводятся  для  материалов,  отмеченных в правом
окне первого пункта меню, либо для группы материалов, отмеченной
в правом окне второго пункта меню. Отметка производится клавишей
"INS".

       4.3. Просмотр показателей преломления и ослабления.
     Длины волн,для  которых  выводятся показатели преломления и
ослабления ,  задаются в окне настройки соответствующего пункта.
Можно  задать  как стандартные длины волн ,  так и в специальном
окне любую длину волны.  Если заданная  длина  волны  лежит  вне
рабочего   интервала   длин   волн,   то   для   соответствующей
характеристики выводится 0.

       4.4. Просмотр дополнительных характеристик.
     В окне  настройки для этого пункта меню необходимо отметить
клавишей "INS" характеристики ,  которые необходимо просмотреть.
Одновременно  можно выводить несколько характеристик при условии
что все характеристики одного материала помещаются на экран.

       4.5. Поиск аналогов.
      Выбор аналогов заданного оптического материала заключается
в нахождении  в  указанном  каталоге  характеристик,  минимально
отличающихся  от  указанных  характеристик  заданного материала.
Поиск может производится по показателю преломления и числу  Аббе
для  длин  волн  "e"  или  "D"  ,  либо  по значениям показателя
преломления для указанных длин волн. Может задаваться вес каждой
из заданных характеристик.
     Задание материала  производится  либо  его   маркой,   либо
значениями  показателя на длинах волн D или e и соответствующими
числами Аббе,  либо значениями показателей на  требуемых  длинах
волн.
     Задание марки    производится    нажатием     "ENTER"     в
соответствующей  строке  окна  настройки  и  последующем  выборе
стекла из появляющегося списка.  Каталог,в котором  производится
поиск, устанавливается в первой строке.
     Выводится 5 ближайших материалов в среднеквадратическом или
средневзвешенном смысле.
     В окне  настройки  в  полях   редактирования   производится
задание  необходимых данных для поиска.  Различаются текстовые и
числовые  поля.  В   текстовых   полях   изменение   данных   на
альтернативное   производится   нажатием   клавиши  "ENTER"  или
"Пробел".

      4.6. Диаграмма стекол.

     Строятся две диаграммы оптических материалов установленного
каталога  -  диаграмма  "показатель  преломления" - "коэффициент
дисперсии" ( число Аббе ) и диаграмма  "  относительная  частная
дисперсия" (P) - " коэффициент дисперсии "( V ).
     Диаграммы строятся для всех материалов  каталога  либо  для
отмеченных материалов.
     На диаграмме идентифицируются оптические материалы  текущей
оптической системы.  Материалы идентифицируются номером среды, в
которой материал встретился в первый раз по ходу луча.
     Для построения  диаграммы  требуется  дополнительный  объем
оперативной  память,  если  свободной   памяти   мало   выдается
сообщение " Мало памяти".
     Если диаграмма строится для "стандартных длин волн", тогда
          "показатель преломления" -   Ne или Nd
          "коэффициент дисперсии" =    (Ne-1)/(NF'-NC')
                              или =    (Nd-1)/(NF -NC )
          "относительная частная дисперсия" -
                                  =   (dN)/(NF'-NC')
                                  =   (dN)/(NF -NC )
       здесь dN- средняя дисперсия для выбранного пользователем
                 участка спектра.
      Выбор длины   волны   (   e  или  d  )  для  интерпретации
коэффициента  дисперсии  и   относительной   частной   дисперсии
производится  во  второй  строке  окна  выбора,  а выбор участка
спектра в третьей  строке.Выбор  производится  нажатием  клавиши
"ENTER" или "SPACE".
      На диаграмме  отмечаются   только   оптические   материалы
заданные марками , при этом текущий каталог может не совпадать с
каталогами стекол использующимися в оптической системе.

     Если диаграмма строится для "длин волн оптической системы",
тогда:
     "показатель преломления" -  для основной длины волны N[Л0]
     "коэффициент дисперсии" =   (N[Л0] - 1)/(N[Лmin] - N[Лmax])
     "относительная частная дисперсия" -
                       =   (N[Лmin] - N[Л0])/(N[Лmin] - N[Лmax])
Здесь: Лmax - максимальная длина волны
       Лmin - минимальная  длина волны

  На диаграмме стекол для текущего материала выводится :
        - марка материала;
        - N - показатель преломления  на основной длине волны;
        - V - число Аббе ( "коэффициент дисперсии ").
        - P-  "относительная частная дисперсия"
     Движение по диаграмме  производится  клавишами  перемешения
курсора.  Любой  участок  диаграммы можно увеличить ,  использую
рамку вызываемую по нажатию клавиши "SPACE".  Движение рамки  по
диаграмме  также  производится  клавишами  перемещения курсора ,
изменение размеров - клавишами "Grey + " и "Grey -".  Увеличение
размеров  проводится  нажатием  клавиши "ENTER".  Восстановление
исходного  вида  диаграммы  -  клавиша  "F5".   Отображение   на
диаграмме марок всех материалов - клавиша "F4".
