
    СИСТЕМА АВТОМАТИЗИРОВАННОГО ПРОЕКТИРОВАНИЯ ОПТИЧЕСКИХ СИСТЕМ.

                              ОПАЛ - ПК

                   РУКОВОДСТВО ДЛЯ ПОЛЬЗОВАТЕЛЕЙ.

            ОПТИМИЗАЦИЯ ЦЕНТРИРОВАННЫХ ОПТИЧЕСКИХ СИСТЕМ.

                        ( работы OPT и OPTW )



                             АННОТАЦИЯ.

  Работы  OPT   и  OPTW   предназначены  для   оптимизации    качества
оптических  систем   с  помощью   автоматической  и   целенаправленной
вариации значений конструктивных параметров.

  Принципиальное  различие  работ  OPT  и  OPTW заключается в способах
оценки качества  аберрационной коррекции.  В работе  OPT такая  оценка
производится  с   помощью  геометрических   (поперечных,   продольных)
аберраций системы, вычисленных для  отдельных лучей в пределах  зрачка
системы  и   для  выбранных   точек  поля   зрения.  В   работе   OPTW
характеристиками   аберрационной   коррекции   являются   коэффициенты
зонального  (для  данных  точек  поля)  полихроматического  разложения
функции волновой аберрации по  ортогональным полиномам от зрачковых  и
хроматических координат.

  Наряду с выполнением основной задачи - повышения качества  коррекции
аберраций проектируемых  систем, в  работах OPT  и OPTW обеспечивается
контроль  граничных  значений   конструктивных  параметров,  а   также
функциональных, эксплуатационных и технологических характеристик.

  ЗАДАЧА оптимизации формулируется как задача поиска такой  комбинации
параметров,  при  которой  оптическая  система  обладала  бы наилучшим
качеством при условии выполнения  заданных ограничений на значения  ее
параметров и характеристик.


                             1.ВВЕДЕНИЕ

  Для   выполнения   процедуры   оптимизации   необходимо,  во-первых,
определить  структуру  критерия  качества  системы, во-вторых, указать
варьируемые параметры  и,  в-третьих, установить, если  нужно, область
допустимых решений, т.е. ввести ограничения.


                        1.1 КРИТЕРИЙ КАЧЕСТВА

  Критерий  качества  -  это  некоторая  неотрицательная  функция   от
параметров  системы,  значение  которой  определяет степень близости к
"идеальному"   решению   и   которая   используется   для    сравнения
промежуточных   вариантов,   автоматически    генерируемых   в    ходе
оптимизации.  Чем  меньше  критерий,  тем  лучше  очередной   вариант.
Наилучшим  вариантом  считается  тот,  который  соответствует минимуму
критерия.  Универсального  машинного   критерия  качества   оптических
систем еще не найдено и его формирование - задача конструктора.

  В работах OPT и OPTW  критерий качества представляется в виде  суммы
квадратов отступлений  от требуемых  значений характеристик  заданного
набора. Этот набор формируется  пользователем и состоит, как  правило,
либо из  аберраций отдельных  лучей, либо  из коэффициентов разложения
аберрационной  функции,  но  может включать  и  другие характеристики,
например, увеличение, задний отрезок, длину системы и проч.

  Для  получения   удовлетворительного  результата   в  общем   случае
недостаточно указать,какие из  характеристик участвуют в  формировании
критерия,   необходимо   определить   их   относительную   роль.   Это
осуществляется   введением   индивидуальных   весовых   множителей   и
масштабных  коэффициентов.  При  вычислении  критерия  разность  между
текущим  и  требуемым  значением  характеристики делится на масштабный
коэффициент  (нормируется)   и  умножается   на  весовой    множитель.
Отмасштабированные  и  взвешенные  разности называются ОПТИМИЗИРУЕМЫМИ
ФУНКЦИЯМИ,  сумма  квадратов  которых  и  рассматривается как критерий
качества системы.

  В работах OPT  и OPTW предусмотрена  система умолчания для  заданных
значений,  весов  и  норм,  но  часто  эти  данные  требуют  уточнения
(отметим,  что  уменьшение  веса  эквивалентно увеличению нормирующего
коэффициента и  наоборот.   Выбор способа  варьирования относительного
вклада отдельных характеристик в критерий предоставлен  пользователю).
Подбор  конкретных  значений  весов  или  норм  в  общем случае задача
сложная и трудоемкая,  выполняемая методом проб.   Ее цель -  это либо
достижение компромисса между  требованиями к значениям  большого числа
характеристик,  которые  одновременно  не  могут  принимать   заданные
значения, либо  получение необходимых  значений других  характеристик,
которые при  оптимизации не  контролируются из-за  неприемлемых затрат
на вычисление или сильной  нелинейности по отношению к  параметрам, но
используются при аттестации системы.

  Например, в  работе OPT  при оптимизации  геометрически ограниченных
систем,  к  которым  относится  большинство  фотообьективов, критерием
качества  аберрационной  коррекции  служит  сумма  квадратов аберраций
отдельных  лучей,  которая  плохо   коррелирует  со  значениями   ЧКХ.
Корреляцию  между  аттестационной  оценкой  и машинным критерием часто
удается  улучшить  варьированием  весовых  множителей  для   аберраций
отдельных  зон  зрачка,  однако  удачный  выбор  весов  требует опыта,
интуиции, а  иногда просто  везения.   Проблемы взаимной  балансировки
аберраций не  возникает, когда  качество оптимизируемой  системы можно
оценивать величиной среднеквадратической деформации волнового  фронта.
При  оптимизации  объективов   для  фотолитографии,   микрообъективов,
визуальных систем достаточно  воспользоваться работой OPTW,  в которой
характеристиками   аберрационной    коррекции   служат    коэффициенты
зонального  полихроматического  разложения  функции волновой аберрации
по ортогональным полиномам  от относительных зрачковых  и спектральных
координат.     Значения    коэффициентов   разложения    автоматически
нормируются так ,  что сумма их  квадратов представляет собой  среднее
значение  квадрата  аберрационной  функции  системы.  Если  в   список
оптимизируемых  функций  включены  коэффициенты  разложения,  машинный
критерий   качества   системы   оказывается   адекватным  объективному
критерию качества изображения.

  Средства  входного  языка  позволяют  описать  набор характеристик в
наглядной и компактной  форме. Характеристики описываются  символьными
марками в  правой части  специального оператора  как элементы массива.
Марка   является   символическим   обозначением   одной   либо  группы
однородных  характеристик  и   состоит  из  двух   частей:   имени   и
указателей.   Имя    определят   тип  характеристики,   а    указатели
конкретизируют номер луча, тип сечения,  степень полинома и т.п.   Имя
и указатели  отделяются символом  /.   Например, поперечная  аберрация
первого луча верхней части меридионального сечения второго  внеосевого
пучка  для  второй  дополнительной  длины  волны  обозначается  маркой
DYM/221U.   Если  указатели  в  марке  отсутствуют,  то символ / можно
опустить.   Для   описания  одной  маркой   целой  группы   однородных
характеристик  используется  символ  *  ,который  интерпретируется как
несколько  цифровых  индексов.   Например,  описание  полного   набора
коэффициентов в списке оптимизируемых функций работы OPTW имеет вид :

    MF(1) = CWZ/****;

  Один  или  несколько  подряд  следующих  символов *, расположенных в
крайней правой  части указателя,  можно опускать.  Поэтому приведенное
описание коэффициентов эквивалентно такому:

    MF(1) = CWZ;

  В   работе   OPT   поперечные   аберрации   лучей   осевого   пучка,
меридионального  и  сагиттального  сечений  внеосевых  пучков   можно
описать следующим образом:

    MF(1-4) = DYA/**,DYM/****,DYS/***,DXS/***;
           или
    MF(1-4) = DYA,DYM,DYS,DXS;

  Построение  критерия   качества  только   на  основе   аберрационных
характеристик,  как  отмечалось  выше,  не  обязательно.   При решении
некоторых  специфических  задач  за  критерий  качества  можно принять
степень  близости  к  заданным  значениям  одной или нескольких других
характеристик оптической  системы. Например,  если в  качестве таковых
взять увеличение, длину,  положение плоскости Гаусса  и параксиального
выходного зрачка системы, описание будет иметь вид :

    MF(1-4)  = VG0,  L,   SG',   SP0';
    MF#(1-4) = 5,    55,   10,      1;
    MFW(1-4) = 1,   0.1,  0.2,    0.1;

    здесь - MF  - список характеристик
            MF# - массив требуемых значений
            MFW - массив весов

  При оптимизации можно использовать  и смешанный критерий, включив  в
него  как   аберрационные,  так   и  другие   характеристики.   Cписок
характеристик,   участвующих   в   формировании   критерия,   является
обязательным  элементом   исходных  данных.    При   описании   прочих
параметров  оптимизируемых  функций  (весовых  коэффициентов  и  норм)
можно   воспользоваться   системой   умолчания.    Заданные   значения
характеристик по умолчанию принимаются  равными нулю, веса -  единице,
а масштабы зависят от  типа характеристик.  Для  продольных аберраций:
0.1  мм  или  дптр,  для  поперечных  аберраций  0.01  мм или градуса.
Коэффициенты  разложения  измеряются  в  единицах основной длины волны
спектрального интервала.

  Итак,  определить   структуру  критерия   -  значит   описать  набор
характеристик,  близость  которых  к  заданным значениям может служить
оценкой  качества  системы,  и  явно  или  с  использованием  принципа
умолчания указать требуемые  значения, веса и  нормы.


                     1.2. ПАРАМЕТРЫ ОПТИМИЗАЦИИ

  Вторым  обязательным  элементом  исходных  данных  является   список
параметров оптимизации.   Выбор параметров,  также как  и формирование
критерия  качества,  -  весьма  ответственная  процедура.   Недостаток
параметров   ограничивает   возможности   поиска   удовлетворительного
решения, а их избыток приводит не только к большим затратам  машинного
времени,  но  и  снижает  эффективность  алгоритмов  отимизации  из-за
повышения вероятности потери точности при больших размерностях  задач.
При   включении   конструктивных   параметров   в   число   параметров
оптимизации следует избегать, во-первых, слабо влияющих переменных,  а
во-вторых,  тех,  которые  при  оптимизации  системы  имеют устойчивую
тенденцию к выходу за пределы интервала допустимых значений.

  В  задании  на  оптимизацию  варьируемые  параметры  перечисляются в
правой   части   специального   оператора   как   элементы  массива  в
соответствии с  общими правилами  входного языка  системы ОПАЛ-ПК. Для
их  идентификации   используются  символьные   марки,   обеспечивающие
возможность   компактного   описания   групп   однородных  параметров.
Структура марок параметров  аналогична структуре марок  характеристик.
Например,  список   параметров,  включающий   кривизны  первых   10-ти
поверхностей  системы,   осевые  промежутки   и  положение   плоскости
установки имеет вид:

    PAR(1-3) = C/1-10, D/1-9, SI;


                   1.3 ОГРАНИЧЕНИЯ ПРИ ОПТИМИЗАЦИИ

  В  принципе,   для  выполнения   процедуры  оптимизации   в  задании
достаточно указать список  характеристик, используемых при  вычисления
критерия (оператор MF)  и список параметров  (оператор PAR).   Это тот
минимум  данных,  без  которых  оптимизация  не  выполняется.   Однако
оптическая  система,  полученная   в  результате  минимизации   только
критерия  качества,  как  правило,   не  будет  удовлетворять   другим
условиям,  которые  в  критерии  не  отражены.  Например, наилучший, с
точки  зрения  структурных  характеристик,  результат  может оказаться
неприемлемым  в  отношении  увеличения  системы, ее габаритов, заднего
отрезка и возможности физической реализации. Включение  дополнительных
условий в оценку  качества системы приводит  к проблеме выбора  весов;
но,  самое   главное,  требования   к  такого   рода   характеристикам
формулируются иначе, чем к  определяющим качество.  О  последних можно
сказать,  что  чем  они  ближе  к  заданным,  тем  лучше.  Но  имеются
характеристики,  для  которых  иногда  необходимо  строгое  выполнение
условия их  равенства вполне  определенным значениям,  причем выход за
пределы достаточно узкого допуска может означать полную  непригодность
системы.   Для  другой  группы  характеристик  могут  быть  определены
границы  сравнительно  широкого  допустимого  интервала,  а   значения
внутри   интервала   безразличны.     Описанные   условия   называются
ограничениями задачи.   В работах OPT  и OPTW допускаются  ограничения
двух типов:  равенства и неравенства.


                    1.3.1. ОГРАНИЧЕНИЯ - РАВЕНСТВА

  Наличие ограничений-равенств  означает, что  в процессе  оптимизации
значения указанных  характеристик не  выходят за  пределы назначенного
допуска. Чаще всего  в качестве равенств  контролируются параксиальные
характеристики:    увеличение,   положение   плоскости   Гаусса,задний
отрезок  либо  габариты.   При  необходимости  с  помощью ограничений-
равенств можно контролировать  аберрации отдельных лучей  (работа OPT)
или коэффициенты разложения функции волновой аберрации  (работа OPTW).
Следует  отметить,   что   включение   в   число  ограничений-равенств
аберрационных характеристик  существенно увеличивает  время выполнения
задания.

  Описание  равенств  включает  список  характеристик,  их   требуемых
значений  и  допустимых  отклонений.  По  умолчанию  заданные значения
принимаются  равными  исходным,  а  величины  допуска  зависят от типа
характеристик. В частности, допуск на увеличение устанавливается =  2%
от номинала.  При описании  равенств достаточно  списка характеристик,
например,  условие  сохранения  исходного  увеличения  с  точностью 2%
(допуск  и  требуемое  значение  принимаются по умолчанию) описывается
оператором вида :

      EQF(1) = VG0;

  Когда требуемые значения  характеристик отличаются от  исходных (они
перечисляются в  операторе EQF#(*)=  ...; ),  перед началом собственно
оптимизации все они "подгоняются"  под заданные с помощью  специальной
процедуры  "восстановления".   Критерий   качества  в  это  время   не
контролируется,  и  вполне  вероятно,  что  он станет больше или будут
нарушены  условия  прохождения  действительных  лучей.   Поэтому   при
использовании  равенств   желательно  поставить   условие   сохранения
исходных  значений  характеристик,  а  не  их "подгонки" под заданные;
"подгонку" же лучше осуществлять с помощью ограничений-неравенств.



  Приведем примеры простейших заданий.

  ПРИМЕР 1. При оптимизации параметрами служат кривизны первых 10-ти
            поверхностей, первые пять осевых расстояний и положение
            плоскости установки. Исходное значение увеличения
            сохраняется неизменным. В качестве оптимизируемых функций
            используется полный набор коэффициентов разложения
            аберрационной функции.

            PAR(1-3) = C/1-10, D/1-5, SI;
            EQF(1) = VG0;
            MF(1) = CWZ;

  ПРИМЕР 2. Параметрами оптимзации служат все кривизны, за
            исключением 5-ой, и первые девять осевых расстояний, за
            исключением 5,6,и 7-ого.  Сохраняются увеличение и
            положение плоскости Гаусса. В качестве оптимизируемых
            функций используются взвешенные и отнормированные
            значения поперечных аберраций лучей осевого пучка,
            меридионального и сагиттального сечений внеосевых
            пучков.  Для аберраций осевого пучка весовой коэффициент
            установлен в 3 раза выше, чем для аберраций внеосевых
            пучков.

            PAR(1-4) = C, D/1-9, -C/5, -D/5-7;
            EQF(1-2) = VG0, SG';
            MF(1-6) = DYA, DYM, DYS, DXS;
            MFW(1) = 3;



                  1.3.2. ОГРАНИЧЕНИЯ - НЕРАВЕНСТВА

  Ограничения-неравенства используются  в  процессе  оптимизации   для
контроля параметров  и характеристик,  значения которых  не могут быть
произвольными, а должны лежать в определенном интервале.


                  1.3.2.1. ОГРАНИЧЕНИЯ НА ПАРАМЕТРЫ

  Ограничения  на  параметры  устанавливаются  из  условий  физической
реализуемости, технологичности и конструктивности системы.   Граничные
значения параметров описываются в задании специальными операторами,  с
помощью  которых  устанавливаются  верхние  (правые)  и нижние (левые)
границы   допустимых   интервалов.    Для   каждого   типа  параметров
(толщины,кривизны  и  т.п.)  существуют  отдельные  операторы описания
границ.   Например,  минимальные  и  максимальные  допустимые значения
толщин и  промежутков записываются  как элементы  массива в операторах
вида:

     D<(1,2,3) = 1,3,1;
     D>(2,5,7) = 5,3,4;


                1.3.2.2. МЕТОДЫ КОНТРОЛЯ ОГРАНИЧЕНИЙ
                         НА ЗНАЧЕНИЯ ПАРАМЕТРОВ

  Для контроля  ограничений на  значения параметров  конструктор может
воспользоваться одним из двух способов.

  Первый  способ,  называемый  методом  исключения, заключается в том,
что, когда в  процессе оптимизации некий  параметр выходит за  пределы
допустимого  интервала,  его  значение  фиксируется и он автоматически
исключается из числа варьируемых.  Включение параметра в список  "дей-
ствующих" также производится  автоматически при исчерпании  других ре-
сурсов оптимизации  оптической системы.   Первый способ  прост, но  не
позволяет использовать  для оптимизации  те конструктивные  параметры,
которые  в  исходной  системе  находились  вне  допустимого интервала.
Если  в   задании  указаны   границы  параметров,   метод   исключения
подключается автоматически.

  ПРИМЕР 3. Дополним задание примера 1. ограничениями на
            значения осевых расстояний:

            PAR(1-3) = C/1-10, D/1-5, SI;
            EQF(1) = VG0;
            D<(1-5) = 0.1;  D>(1,3) = 3;
            MF(1) = CWZ;

  Второй способ основан  на методе модифицированных  штрафных функций.
Суть  метода  заключается  в  том,  что  при  нарушении  неравенств  к
критерию  качества  добавляется  штраф,  величина  которого  монотонно
возрастает  по  мере  удаления  от  допустимой  области. Одновременная
минимизация   оценочной   и   штрафной   функций   позволяет  добиться
соблюдения неравенств  в пределах  установленных допусков.  Сходимость
процесса  обеспечивается  за  счет  специальных  переменных  "веса"  и
"сдвига".  Значения  этих  переменных  пересчитываются   автоматически
после выполнения определенной  группы шагов ("цикла")  решения задачи.
Второй  способ  позволяет  использовать  для оптимизации те параметры,
которые в  исходной системе  не укладываются  в допустимые  интервалы.
Чтобы применить  метод штрафных  функций для  контроля ограничений  на
значения  конструктивных  параметров,  необходимо  описать их в списке
ограничений-неравенств.  Марки  параметров указываются в  правой части
специального оператора - INF.

  ПРИМЕР 4. Воспользуемся методом штрафных функций для контроля 1-го,
            2-го и 3-го осевого расстояний при оптимизации системы
            примера 3.

            PAR(1-3) = C/1-10, D/1-5, SI;
            EQF(1) = VG0;
            D<(1-5) = 0.1;  D>(1,3) = 3;
            INF(1) = D/1-3
            MF(1) = CWZ;


            1.3.3. ОГРАНИЧЕНИЯ НА ЗНАЧЕНИЯ ХАРАКТЕРИСТИК

  Ограничения-неравенства на  значения характеристик  в работах  OPT и
OPTW  контролируются  только   методом  штрафных  функций.    Описание
ограничений  должно  включать,  во-первых,  список  характеристик   и,
во-вторых,  список  их  граничных  значений.   Марки характеристик, на
которые  устанавливаются  ограничения,  указываются  в  правой   части
оператора  INF   (вместе  с   марками  параметров,   когда   последние
контролируются  методом  штрафных  функций).   Для  описания граничных
значений характеристик  используются специальные  операторы, в  правой
части которых  граничные значения  описываются как  элементы массива с
индексами, соответствующими  индексам  марок  в  операторе  INF.   Для
описания нижних (левых) и  верхних (правых) границ применяются  разные
операторы:  INF< и INF>

  ПРИМЕР 5. Введем в задание примера 4. описание ограничения на длину
            системы (марка L) и положение плоскости Гаусса. Длину
            системы ограничим сверху (справа) значением 30 мм, а
            положение плоскости Гаусса - снизу (слева) значением 1.5 мм

            PAR(1-3) = C/1-10, D/1-5, SI;
            EQF(1) = VG0;
            D<(1-5) = 0.1;  D>(1,3) = 3;
            INF(1-3) = D/1-3, L, SG';
            INF>(2) = 30;  INF<(3) = 1.5;
            MF(1) = CWZ;

  Основная  проблема,  возникающая  при  использовании метода штрафных
функций  -   взаимная  балансировка   штрафов  за   нарушение   границ
нескольких характеристик и их общее соотношение с критерием  качества.
"Слабые"  штрафы   могут  оказаться   незаметными  на   фоне  критерия
качества.   Наоборот, при  "сильных" штрафах  решение, удовлетворяюшее
ограничениям-неравенствам,  может  получиться  неоптимальным  с  точки
зрения критерия качества.  В работах OPTW и OPT исходная  балансировка
всех функций  осуществляется за  счет подбора  специальных нормирующих
множителей.   Значения  нормирующих  коэффициентов  устанавливаются по
умолчанию  в  зависимости  от   типа  контролируемого  параметра   или
характеристики,   но   могут   быть   заданы   конструктором.  В  ходе
оптимизации производится автоматическая  пере- балансировка функций  с
использованием  "внутренних"  переменных  "веса"  и  "сдвига"  с таким
расчетом,  чтобы   в  результате   процедуры  критерий   качества  был
минимальным, а  все ограничения  соблюдались. Обычно    результирующее
решение  соответствует  пересечению  границ  действующих  ("активных")
ограничений.



                   2. СТРУКТУРА ДАННЫХ ОПТИМИЗАЦИИ


                 2.1 ОПИСАНИЕ ВАРЬИРУЕМЫХ ПАРАМЕТРОВ

  Варьируемые  параметры,  т.е.   конструктивные  параметры   системы,
которые могут изменяться в процессе оптимизации описываются с  помощью
оператора вида:

      PAR(*) = пр1,пр2...прn;
            или
      PAR(к1-к2) = прк1,...,прк2;
            или
      PAR(к1,к2) = прк1,прк2;

где:  n - количество марок варьируемых параметров (целое число),
      к1,к2  -  порядковые номера марок в массиве PAR;
      пр1,пр2,...,прк1,прк2,прn - марки параметров

  В  качестве  варьируемых  параметров  могут  использоваться кривизны
поверхностей,  толщины,  осевые  промежутки, квадраты эксцентриситетов
асферических  поверхностей,  коэффициенты  асферики  высшего  порядка,
коэффициенты  деформации  ГОЭ,  передний  отрезок,  задний  отрезок  и
положение  апертурной  диафрагмы.  Марки  перечисленных  параметров  и
правила  их  записи  приведены  в  разделе  7.1 ( таблица 1) настоящей
инструкции.
  Общее количество параметров не должно превышать 64.

  ПРИМЕР 6. Пусть необходимо оптимизировать проекционную оптическую
            систему, состоящую их двух симметричных двухлинзовых
            склеенных объективов. В общем случае в качестве
            варьируемых параметров могут быть использованы: положение
            апертурной диафрагмы, кривизны, толщины, передний
            отрезок. Заметим, что в силу симметрии значения
            конструктивных параметров второй части системы не
            независимы и в операторе PAR не указываются.

            PAR(1-4) = SD, C/1-3, D/1-3, SO;


                    2.2. СВЯЗИ МЕЖДУ ПАРАМЕТРАМИ.

  В процессе оптимизации между конструктивными параметрами могут  быть
выдержаны определенные связи вида:

      пз = кс * пс

где   пз,пс - значения связанного(зависимого) и свободного
             (независимого) параметров соответственно ;
      кс - коэффициент связи.

  Связи выдерживаются для следующих пар параметров:
      радиус - радиус;
      радиус - толщина;
      толщина - положение диафрагмы;
      передний отрезок - задний отрезок;
      передний отрезок - радиус первой поверхности;
      задний отрезок - радиус последней поверхности.

  Описание связей включает:
      список марок свободных параметров;
      список марок связанных параметров;
      список коэффициентов связей (массив чисел).

  Марки свободных параметров определяются оператором:

      PARF(*) = пс1,пс2,...,псn;
             или
      PARF(к1-к2) = пск1,...,пск2;
             или
      PARF(к1,к2) = пск1,пск2;

где   n - общее число марок свободных параметров;
      пс1,пс2,пск1,пск2,псn - марки свободных параметров

  Марки связанных  параметров  указываются  в  строгом  соответствии с
расположением соответствующих им марок свободных параметров с  помощью
оператора:

      PARC(*) = пз1,пз2,...,пзn;
             или
      PARC(к1-к2) = пзк1,...,пзк2;
             или
      PARC(к1,к2) = пзк1,пзк2;

где   n - общее число марок связанных параметров;
      пз1,пз2,пзк1,пзк2,пзn - марки связанных параметров

  Коэффициенты   связи   записываются   в   строгом   соответствии   с
расположением  марок  свободных  и  связанных  параметров в операторах
PARF и PARC с помощью оператора:

      CC(*) = кс1,кс2,...,ксn;
           или
      CC(к1-к2) = кск1,...,кск2;
           или
      CC(к1,к2) = кск1,кск2;

где   n - общее число коэффициентов связи;
      кс1,кс2,кск1,кск2,ксn - коэффициенты связи между пс1 и пз1,
      пс2 и пз2, пск1 и пзк1, пск2 и пзк2, псn и пзn соответственно.

  ВНИМАНИЕ !
      - При описании связей все три оператора - PARF,PARC,CC
        являются ОБЯЗАТЕЛЬНЫМИ.
      - Свободные параметры, указанные в списке PARF, должны
        быть описаны в операторе PAR
      - Связанные параметры в операторе PAR не указываются.
        Общее число свободных параметров не должно превышать 64.

  ПРИМЕР 7. Для получения симметричной конструкции,описанной в
            примере 6, необходимо выдержать связь между
            конструктивными параметрами первого и второго склеенных
            компонентов. Связи можно указать отдельно для каждой пары
            связанных параметров:

            PAR(1-4) = SD, C/1-3, D/1-3, SO;
            PARF(1-5) = C/1, C/2, C/3, D/1, D/2;
            PARC(1-5) = C/6, C/5, C/4, D/4, D/5;
            CC(1-3) = -1;
            CC(4-5) = 1;

или, в более компактной форме, для групп однородных параметров :

            PAR(*) = SD, C/1-3, D/1-3, SO;
            PARF(*) = C/1-3, D/1-2;
            PARC(*) = C/6-4, D/5-4;
            CC(*) = -1,1;


                    2.3 ОГРАНИЧЕНИЯ НА ПАРАМЕТРЫ

    Ограничения на  параметры можно  описать двумя  способами: либо  с
помощью  специализированных  операторов  описания  границ   параметров
определенного   типа,   либо   универсальными   операторами  граничных
значений     параметров     и      характеристик     при      описании
ограничений-неравенств.   Первый   способ   позволяет   применить  для
контроля  параметров  как  метод  исключения,  так  и  метод  штрафных
функций.  Второй  способ  предполагает  использование  метода штрафных
функций. Ниже приводятся специализированные операторы описания  границ
параметров  отдельных  типов.  С  их  помощью  можно описать граничные
значения  кривизн  поверхностей,  осевых  расстояний, эксцентриситетов
асферических  поверхностей,  а  также  смещения  плоскости  установки,
когда  последнее  используется  в  качестве параметра оптимизации. Для
описания  ограничений  на  значения  конструктивных  параметров служат
следующие операторы:

 - граничные значения кривизн поверхностей

      C<(J1,J2,..Jк) = CMINJ1,CMINJ2...CMINJк;
      C>(I1,I2,..Iк) = CMAXI1,CMAXI2...CMAXIк;

где   J1..Jк, I1..Iк - номера повехностей
      CMINJ1,..CMINJк - минимальные значения кривизн J1-ой..Jк-ой
                        поверхностей;
      CMAXI1,..CMAXIк - максимальные значения кривизн I1-ой..Iк-ой
                        поверхностей.

      ПРИМЕЧАНИЕ. Помимо описаний типа C<(J1,J2,..Jк) ,в левой части
                  операторов определения граничных значений на
                  конструктивные параметры могут использоваться и
                  другие описания, допускаемые общими правилами
                  языка ОПАЛ-ПК, например, C<(J1-Jk).  Однако для
                  сокращения объема инструкции все возможные варианты
                  не рассматриваются.

 - граничные значения осевых расстояний

      D<(J1,..Jк) = DMINJ1,...DMINJк;
      D>(I1,..Iк) = DMAXI1,...DMAXIк;

где   J1,..Jк, I1,..Iк - номера осевых расстояний;
      DMINJ1,DMINJк - минимальные значения J1-го и Jк-го
                      осевых расстояний;
      DMAXI1,DMAXIк - максимальные значения I1-го и Iк-го
                      осевых расстояний;

 - граничные значение эксцентриситетов асферических поверхностей

      ES<(J1,...Jк) = е2MINJ1,...е2MINJк;
      ES>(I1,...Iк) = е2MAXI1,...е2MAXIк;

где   J1,..Jк, I1,..Iк - номера поверхностей;
      е2MINJ1,е2MINJк - минимальные значения квадратов
                        эксцентриситетов J1-ой и Jк-ой поверхностей;
      е2MAXI1,е2MAXIк - максимальные значения
                        квадратов эксцентриситетов I1-й и Jк-й
                        поверхностей;

 - граничные значения переднего отрезка

      SO< = минимальная  величина переднего отрезка;
      SO> = максимальная величина переднего отрезка;

 - граничные значения заднего отрезка

      SI< = минимальная величина заднего отрезка;
      SI> = максимальная величина заднего отрезка;

  ВНИМАНИЕ !!!
  1. Для  всех перечисленных  выше операторов  определение типа границ
(MAX или MIN)  ведется по алгебраическому  значению, т.е. -100  меньше
+1;  -1  больше -10

  2. На толщины линз  и промежутки устанавливаются границы  физических
значений  параметров,  не  зависимо  от  направления   распространения
света,  т.е.минимальная  толщина   физически  реализуемой  линзы   или
промежутка, заданная  оператором D<  всегда положительна.   Фактически
при  оптимизации  контролируется  величина:   D*N/модуль(N),  где  D -
осевое расстояние, N - показатель преломления со знаками, принятыми  в
оптике.

  3. Ограничения на  параметры, установленные с  помощью перечисленных
операторов, можно контролировать  двумя способами: методом  исключения
и методом штрафных  функций. Метод исключения используется только  для
независимых  параметров.  Он  подключается  автоматически,  если   для
параметра  указана  хотя  бы   одна  граница.  Обязательным   условием
применения  метода  исключения   является  выполнения  ограничений   в
исходной  системе.   Метод   штрафных  функций  более  трудоемок,   но
универсален.   Его  можно  использовать  для  контроля  независимых  и
связанных параметров, причем условие выполнения ограничений в исходной
системе не  обязательно. Чтобы  применить метод  штрафных функций  для
контроля   параметров,   необходимо   записать   их   марки  в  списке
ограничений-неравенств.   Границы  параметров  в  операторах  описания
допустимого  интервала  ограничений-неравенств   можно  при  этом   не
указывать, если они описаны специальными операторами.

  4.  Ограничения  на  величину  заднего отрезка, заданные операторами
SI>  и  SI<,  учитываются  только  в  том случае, когда задний отрезок
является независимым или связанным параметром оптимизации, т.е.  когда
он указан в  списках PAR или  PARC. Как независимый  параметр SI можно
контролировать  либо   методом  исключения,   либо  методом   штрафных
функций.  Когда  SI  -  связанный  параметр, он может контролироваться
только  методом  штрафных  функций.  Если  задний отрезок определяется
положением    плоскости    Гаусса,    т.е.    является   параксиальной
характеристикой  системы,  в  списке  параметров  он  не указывается и
может контролироваться  как ограничение  общего вида  - равенство  или
неравенство,   либо   как   оптимизируемая   функция.   При  этом  для
идентификации  положения  плоскости  Гаусса  используется  специальное
обозначение - марка SG'. Границы допустимого интервала либо  требуемое
значение  отрезка  указываются  в  универсальных  операторах  описания
граничных или заданных значений  характеристик в соответствии с  типом
используемых функций.

  ПРИМЕР 8. Для системы, описанной в примере 1, потребуем, чтобы
            радиус кривизны первой поверхности был не менее 50мм по
            абсолютной величине, что соответствует изменению ее
            кривизны от -0.02 до 0.02, 1-е, 2-е и 3-е осевые
            расстояния находились в диапазонах 10мм - 20мм,
            5мм - 8мм, 1мм - 40мм соответственно, передний отрезок
            был не менее 100 мм (предмет слева от оптической
            системы). Названные выше ограничения могут быть записаны
            следующим образом:

            C<(1)=-0.02;   C>(1)=0.02;
            D<(1-3)=10.,5.,1;   D>(1-3)=20.,8.,40.;
            SO>=-100;


          3. ОПИСАНИЕ ТРЕБОВАНИЙ К ЗНАЧЕНИЯМ ХАРАКТЕРИСТИК

  Для  управления  значениями   характеристик  оптической  системы   в
процессе оптимизации используются три типа функций:

      - оптимизируемые функции
      - функции-равенства
      - функции-неравенства

  Каждый   тип   функций   отличается   назначением,   видом  связи  с
характеристикой  системы  и  способом  контроля при оптимизации. Выбор
типа  функции  для  контроля  определенной характеристики осуществляет
конструктор.


                    3.1. ОПТИМИЗИРУЕМЫЕ ФУНКЦИИ.

  Оптимизируемые  функции  используются  для  контроля  характеристик,
близость значений которых к заданным может служить критерием  качества
системы.  Оптимизируемая  функция  определяется  как  разность   между
текущим   и   требуемым   значениями   характеристики,   деленная   на
нормирующий коэффициент и умноженная на весовой множитель:

                      офт(I) - офз(I)
              оф(I)= ───────────────── * офв(I)
                           офн(I)

где   оф(I)  - значение оптимизируемой функции
      офт(I) - текущее значение характеристики
      офз(I) - заданное значение характеристики
      офв(I) - весовой множитель
      офн(I) - норма (единица измерения)
      I      - порядковый номер характеристики
               в списке оптимизируемых функций

  Качество  системы  характеризуется  суммой  квадратов оптимизируемых
функций.


               3.1.1. ОПИСАНИЕ ОПТИМИЗИРУЕМЫХ ФУНКЦИЙ

  В общем случае описание оптимизируемых функций включает:

      - список характеристик,
      - список требуемых значений характеристик,
      - список весовых  коэффициентов,
      - список нормирующих коэффициентов.

  Список  характеристик  является   обязательным  элементом   описания
оптимизируемых функций. Для  прочих параметров оптимизируемых  функций
предусмотрены значения  по умолчанию,  но при  необходимости они могут
быть  заданы  конструктором.   Требуемые  значения  характеристик   по
умолчанию принимаются  равными нулю,  весовые коэффициенты  - единице.
значения нормирующих коэффициентов, принятые по умолчанию, зависят  от
типа  характеристик.  Поперечные  аберрации,  в частности,измеряются в
сотых  долях  мм  или  градуса  для  близкого и удаленного изображения
соответственно.

  Список характеристик задается оператором MF (синоним OF) вида:

      MF(*) = ф1,ф2,...,фn;
           или
      MF(к1-к2) = фк1,...,фк2;
           или
      MF(к1,к2) = фк1,фк2;

где:  ф1..,фк1,...,фк2,фn     -марки характеристик
      n      - количество марок в списке  MF. Общее количество
               марок должно быть меньшим 200.
      к1,к2  - порядковые номера марок в списке MF

  Требуемые значения характеристик указываются в операторе MF# :

      MF#(*) = фз1,фз2,...,фзn;
            или
      MF#(к1-к2) = фзк1,...,фзк2;
            или
      MF#(к1,к2) = фзк1,фзк2;

где:  фз1,..,фзк1,..,фзк2,фзn - заданные значения характеристик;
      n     - общее количество элементов в списке офз;
      к1,к2 - порядковые номера в списке офз.

  Весовые коэффициенты описываются оператором MFW:

      MFW(*) = фв1,фв2,...,фвn;
            или
      MFW(к1-к2) = фвк1,...,фвк2;
            или
      MFW(к1,к2) = фвк1,фвк2;

где   фв1,...фвк1,..,фвк2,фвn - весовые коэффициенты оптимизируемых
функций

  Нормы оптимизируемых функций указываются в операторе MFS:

      MFS(*) = фн1,фн2,...,фнn;
            или
      MFS(к1-к2) = фнк1,...,фнк2;
            или
      MFS(к1,к2) = фнк1,фнк2;

где   фн1...,фн2,фнк1,...фнк2,фнn - нормы характеристик
(положительные действительные числа )

  Марки  характеристик  в  списке  MF  могут  быть  индивидуальными  и
групповыми.  Между элементами  операторов MF,MF#,MFW и MFS  существует
взаимно   однозначное   соответствие,    т.е.   для    характеристики,
обозначенной  J-ой  маркой  в  списке  MF,  устанавливается   заданное
значение MF#(J), нормирующий коэффициент MFS(J) и весовой  коэффициент
MFW(J).  В  случае  групповой  марки  параметры MF#(J),MFW(J) и MFS(J)
устанавливаются для всех характеристик данной группы.

  ПРИМЕР 9. Пусть качество системы характеризуется поперечными
            аберрациями осевого пучка, продольными аберрациями
            главных лучей и дисторсией на краю поля, причем
            значение последней должно быть близким к +5%.

            MF(1-4) = DYA, ZM, ZS, DIS%/1;
            MF#(4) = 5;

  Здесь  мы   воспользовались  системой   умолчания  для    назначения
требуемых  значений  поперечных  аберраций,  а  также норм и весов для
всех характеристик.

  Предположим,  что   в  результате   оптимизации  степень   коррекции
аберрации осевого пучка  оказалась недостаточной, но  можно попытаться
ее улучшить  за счет  ухудшения коррекции  аберраций внеосевых пучков.
Это можно сделать одним из следующих способов:

   а) увеличением веса аберраций осевого пучка
      MF(1-4) = DYA, ZM, ZS, DIS%/1;
      MF#(4) = 5;
      MFW(1) = 10;

   б) уменьшением норм аберраций осевого пучка
      MF(1-4) = DYA, ZM, ZS, DIS%/1;
      MF#(4) = 5;
      MFS(1) = 0.001;

   в) уменьшением веса аберраций внеосевых пучков
      MF(1-4) = DYA, ZM, ZS, DIS%/1;
      MF#(4) = 5;
      MFW(2-4) = 0.1;

  При отсутствии ограничений-неравенств  результат оптимизации в  всех
трех  случаях  будет  одинаковым.  Заметим,  что  абсолютное  значение
критерия качества в  последнем варианте отличается  от его значения  в
первых двух.

  Предположим теперь, что в результате оптимизации по данным  варианта
а)    неудовлетворительной    оказалась     коррекция    хроматических
аберраций  осевого  пучка,  но  ее  можно  улучшить  за счет остальных
аберраций. Увеличим вклад хроматических аберраций в критерий качества:

      MF(1-4) = DYA, ZM, ZS, DIS%/1;
      MF#(4) = 5;
      MFW(1) = 10;
      MF(5) = DYA/'*;  MFW(5) = 20;

  Мы добавили в список  характеристик марку DYA/'*, которая  описывает
группу аберраций первичного хроматизма лучей осевого пучка, и  указали
для аберраций  этой группы  новое значение  весового коэффициента =20.
Ранее аберрации этой группы  неявно были описаны маркой  DYA совместно
с  аберрациями   основной  длины   волны  и   аберрациями   вторичного
хроматизма,  с  весовыми  коэффициентами  =10.  Эквивалентное описание
имеет вид:

      MF(1-4) = DYA, DYA/', ZM, ZS, DIS%/1;
      MF#(5) = 5;
      MFW(1-2) = 10,20;

  Два последних примера иллюстрируют следующее правило: если в  списке
характеристик   встречается   марка,   которая   идентифицирует   одну
характеристику  или  группу  характеристик,  которые  явно или неявно
были  описаны  предыдущими  марками  списка,  то  всем характеристикам
данной группы приписываются  новые значения параметров  оптимизируемых
функций,  указанные  в  операторах  MF#,MFW  и  MFS  и  имеющие тот же
порядковый номер, что и  данная марка. Если соответствующие  параметры
не указаны, то сохраняется их прежнее значение.


                      3.2  ФУНКЦИИ - РАВЕНСТВА

  Характеристики  системы,  которые  в  процессе  оптимизации   должны
сохраняться  в  пределах  заданного  допуска, контролируются с помощью
функций-равенств, или ограничений-равенств. Такие функции имеют вид:

      фр(I) = (рвз(I)-рвт(I)) / рвн(I)

где   фр(I) - значение I-й функции-равенства;
      рвт(I) - текущее значение I-й характеристики;
      рвз(I) - заданное значений I-й характеристики;
      рвн(I) - норма (единица измерения) I-й функции-равенства.

причем: -рвд(I) <= (рвз(I)-рвт(I) <= рвд(I)

где   рвд(I) - положительный допуск на значение I-й характеристики.

  Набор   характеристик,   которые   необходимо  контролировать    как
функции-равенства, задается оператором вида:

      EQF(*) = р1,р2...рn;
            или
      EQF(к1-к2) = рк1,...,рк2;
            или
      EQF(к1,к2) = рк1,рк2;

где   n - общее число функций-равенств (целое число к<=40.)
      р1,р2,рк1,рк2,ркn - марки функций-равенств с порядковыми
      номерами р1,р2,рк1,рк2,ркn соответственно.
      Общее число функций-равенств не должно превышать 20.

  В   качестве   функций-равенств   в   процессе   оптимизации   могут
контролироваться  специальные  характеристики,  а также характеристики
качества аберрационной коррекции.

  ПРИМЕР 10. Пусть для системы, рассмотренной в примере 1,
             конструктор желает контролировать обобщенное увеличение
             всей системы и силу первого компонента с помощью
             функций-равенств.  В этом случае список функций-равенств
             должен выглядеть следующим образом

             EQF(1-2) = VG0, OP/1-3;


                 3.2.1 ФОРМИРОВАНИЕ ФУНКЦИЙ-РАВЕНСТВ

  Для  правильного  формирования  функций-равенств  необходимо задание
следующих дополнительных данных:

 - заданные значения характеристик определяются с помощью
   оператора вида:

      EQF#(*) = рз1,рз2,...,рзn;
             или
      EQF#(к1-к2) = рзк1,...,рзк2;
             или
      EQF#(к1,к2)=рзк1,рзк2;

где   рз1,рзn - заданные значения 1-й, 2-й и n-й характеристик
      рзк1,рзк2 - заданные значения к1-й и к2-й характеристики.

 - нормы (единицы измерения) функций-равенств задаются с помощью
   оператора вида:

      EQFS(*) = рн1,рн2,..,рнn;
             или
      EQFS(к1-к2) = рн1,...,рн2;
             или
      EQFS(к1,к2) = рнк1,рнк2;

где   рн1,рн2,рнn - нормы функций-равенств 1-й,2-й и n-й
      характеристики; рнк1,рнк2 - нормы функций-равенств к1-й и к2-й
      характеристики.


                     3.3  ФУНКЦИИ - НЕРАВЕНСТВА

  Параметры и характеристики оптической системы, для которых  известны
верхние  и   нижние  границы   изменения,  контролируются   с  помощью
функций-неравенств,  или  ограничений-неравенств.  Такие  функции
имеют вид:

      нр<(I) =  ( нрт(I)-нрг<(I) ) / нрн<(I)
      нр>(I) =  ( нрг>(I)-нрт(I) ) / нрн>(I))

где   нр<(I),нр>(I) -   функции-неравенства для нижней и верхней
                        границ изменения I-го параметра или
                        характеристики;
      нрг<(I),нрг>(I) - нижняя и верхняя границы изменения I-го
                        параметра или характеристики;
      нрн>(I),нрн<(I) - нормы (единицы измерения) функций-неравенств
                        для I-го параметра или характеристики;

      нрт(I) -        текущее значение I-го параметра или
                      характеристики;
      тет(I),GIN(I) - внутренние параметры, значения которых
                      автоматически определяются в ходе оптимизации
                      так, чтобы выполнились соотношения:
                      В результате оптимизации (но не обязательно в
                      ходе) текущее значение характеристики нрт(I)
                      должно удовлетворять соотношениям:

      ( нрт(I)-нрг<(I) ) >= -нрд<(I)
      ( нрг>(I)-нрт(I) ) >= -нрд>(I)

где   нрд<(I),нрд>(I) - положительные допуски на нижние и верхние
                        границы изменения значения нрт(I).

  С целью достижения указанных соотношений на базе функций-неравенств
формируются специальные штрафные функции вида:

      шф<(I)= min(  0; ( нр<(I)-t<(I) ) * g<(I)  )
      шф>(I)= min(  0; ( нр>(I)-t>(I) ) * g>(I)  ),

где   шф<(I),шф>(I) - штрафные функции для нижней и верхней границ
                      изменения I-го параметра или характеристики;

      min - функция минимума, которая равна минимальному значению из
            входящих в нее операндов;

      t<(I),t>(i),g<(I),g>(I) - переменные "сдвига" и "веса" соответ-
      ственно для нижней и верхней границ изменения I-го параметра
      или характеристики. Значения переменных "сдвига" и "веса"
      подбираются автоматически, причем их значения сохраняются
      постоянными в пределах группы шагов оптимизации ("цикла").

   Набор параметров  и характеристик,  которые можно  контролировать с
помощью  ограничений-неравенств,  определяется  с  помощью   оператора
вида:

      INF(*) = нр1,нр2,...,нрn;
            или
      INF(к1-к2) = нрк1,...,нрк2;
            или
      INF(к1,к2) = нрк1,нрк2;

где   n - общее количество параметров и характеристик,
          контролируемых с помощью ограничений-неравенств;
      нр1,нр2,нрn - марки 1-го, 2-го и n-го параметров или
                    характеристик;
      нрк1,нрк2 - марки к1-го и к2-го параметров или характеристик.


  ПРИМЕР 11: Пусть для системы, описанной в примере 1, конструктор
             желает контролировать осевые расстояния, толщины по
             краю, а также общий габарит системы (расстояние от
             плоскости предметов до плоскости изображений) с помощью
             функций-неравенств. В этом случае список
             функций-неравенств будет иметь вид:

             INF(1-5) = D/1, D/2, D/3, DE, LSS';


              3.3.1  ФОРМИРОВАНИЕ ФУНКЦИЙ - НЕРАВЕНСТВ

  Для   полного   описания   функций-неравенств   необходимо   задание
следующих дополнительных данных:

 - верхние и нижние границы изменения параметров и характеристик
   описываются с помощью операторов вида:

      INF>(1-к) = нг>1,нг>2...нг>к;
      INF<(1-к) = нг<1,нг<2...нг<к;

где   нг>1,нг<1 - верхняя и нижняя границы изменения 1-го
                  параметра или характеристики;
      нг>к,нг<к - верхняя и нижняя границы изменения к-го
                  параметра или характеристики.

  ПРИМЕЧАНИЕ 1: Если с помощью функций-неравенств контролируются
                параметры, ограничения на которые были заданы с
                помощью операторов типа С<(...)=...; D>(...)=...;
                то определение границ изменения операторами
                вида INF> и INF< не обязательно.

  ПРИМЕЧАНИЕ 2: При необходимости контроля толщин "по краю" с помощью
                функций-неравенств задается только нижняя граница их
                изменения оператором:

      DE<(*) = Dк1,Dк2...DкJ;
      DE<(1-J) = Dк1,Dк2...DкJ;
      DE<(J1,J2) = DкJ1,DкJ2;

где   J,J1,J2 - номера осевых промежутков;

      Dк1,DкJ - минимальные значения 1-й и J-й толщин "по краю";
      DкJ1,DкJ2 - минимальные значения J1-й и J2-й толщин "по краю"


  ПРИМЕР 12: Пусть толщины "по краю", описанные в списке неравенств
             примера 8, должны быть не менее 2 мм и 3 мм соответственно.
             В этом случае наложенные ограничения могут быть записаны:

             DE<(1,3) = 2,3;

 - Допуски на границы изменения характеристик, контролируемых с
помощью ограничений-неравенств, определяются операторами вида:

      INFT(*) = нд1,нд2...ндк;
             или
      INFT(1-к) = нд1,нд2...ндк;

где  нд1,нд1 - допуски на верхнюю и нижнюю границы из-
               менения 1-й характеристики;
     ндк,ндк - допуски на верхнюю и нижнюю границы из-
               менения к-й характеристики;

  ПРИМЕР 13: Пусть имеется список функций-неравенств, соответствующий
             примеру 8. Необходимо выдержать ограничения на осевые
             расстояния, толщины по краю, а общая длина системы
             должна находиться в диапазоне от 150 мм до 170 мм.
             Допуски на изменение осевых расстояний и толщин по краю
             примем =0.1, а на общую длину =1. В этом случае задание
             может иметь вид:

             INF(1-5) = D/1, D/2, D/3, DE, LSS';
             INF>(1-3) = 20,8,40;  INF>(5)=170;
             INF<(1,2,3,5) = 10,5,0,150;
             INFT(1-4)=0.1; INFT(5)=1;

  В том случае, когда ограничения на осевые расстояния уже были
заданы, как в примере 8, границы изменения можно задать только для
общего габарита системы, т.е.:

          INF>(5)=170; INF<(5)=150;


                 4. МАРКИ ПАРАМЕТРОВ И ХАРАКТЕРИСТИК

  Символическая марка в  общем случае представляет  собой совокупность
символов  и  чисел,  состоящую  из  фиксированной  (имя)  и переменной
(указатели) частей.  Эти части  разделяются между  собой символом "/".
Имя  марки  записывается  всегда  слева  от черты и идентифицирует тип
изменяемого или контролируемого параметра (радиусы, осевые  расстояния
и т.п),  либо тип  контролируемой характеристики  (увеличение,отрезок,
аберрация  луча,  коэффициент  разложения  и  т.д.).  Переменная часть
марки,   в   зависимости   от   типа   параметра  или  характеристики,
конкретизирует  номер  поверхности,  номер  луча, тип сечения, степень
переменной и т.п.  Если указатели в  марке отсутствуют, то  символ "/"
можно  опустить.  Для  описания  одной  маркой целой группы однородных
характеристик используется  символ "*"  ,который интерпретируется  как
несколько  цифровых  индексов.  Один  или  несколько  подряд следующих
символов "*",  расположенных в  крайней правой  части указателя, можно
опускать.   Для переменных,  имеющих значение  по умолчанию (например,
тип  предмета),  в  качестве  указателя  может быть использован символ
"?".  Такой указатель соответствует значению по умолчанию.

                       4.1. МАРКИ ПАРАМЕТРОВ.

  Марки параметров используются:

  - в списке марок параметров оптимизации
  - при описании связей между параметрами
  - при описании ограничений-неравенств, если параметры
    контролируются методом штрафных функций.

  Символические имена параметров приведены в таблице 1

                                       таблица 1
  ┌───────┬────────────────────────────────────┐
  │  имя  │      конструктивный параметр       │
  ├───────┼────────────────────────────────────┤
  │  C    │  Кривизны                          │
  │  D    │  Толщины                           │
  │  ES   │  Эксцентриситет                    │
  │  CA   │  Коэффициенты асферики             │
  │       │  Параметры ГОЭ :                   │
  │  CHL  │      кривизна слева                │
  │  CHR  │      кривизна справа               │
  │  WLH  │      длина волны записи            │
  │  CH   │      коэффициенты высшего порядка  │
  │  SO   │  Передний отрезок                  │
  │  SI   │  Задний отрезок                    │
  │  SD   │  Положение диафрагмы               │
  └───────┴────────────────────────────────────┘

  В  переменной  части  марки  параметра,  справа от черты указываются
номера  поверхностей,  промежутков,   параметры  которых   изменяются.
Каждая  марка  может  идентифицировать  один  параметр (индивидуальная
марка) или группу параметров (групповая марка).

  Индивидуальная марка имеет вид: имя/NN,

где имя - идентификатор варьируемых параметров,
          допустимые имена параметров приведены в таб.1
    NN  - номер параметра (номер поверхности, номер осевого
          расстояния); NN-целое положительное число.

  Групповая марка имеет вид: имя/N1-N2,

где имя - идентификатор группы однородных параметров.
    N1 - номер первого изменяемого параметра группы,
    N2 - номер последнего изменяемого параметра группы,
    "-"  - символ-разделитель.

  Имена  групп  совпадают  с  именами  индивидуальных марок параметров
(табл.1).  Групповая  марка  автоматически  расшифровывается как набор
марок параметров с индексами, принимающими  все значения от N1 до  N2.
Если  группа  параметров  охватывает   всю  оптическую  систему   (все
кривизны,все толщины и т.п.),  то индексы N1 и  N2 можно не писать;  в
этом случае марка будет содержать только соответствующее имя.

  Марки SO,SI,SD записываются без указателей.

  Марки  могут  указываться  со  знаком  "-"  перед  их  символическим
именем, что  означает исключение  параметра или  группы параметров  из
набора, описанного предыдущими марками.

  ПРИМЕРЫ:

  1. PAR(1-4) = C/1-18, -C/11-14, D/6, SD;

     В качестве параметров оптимизации описаны радиусы с 1-го  по
    10-ый и с 15-го по 18-ый, 6-я толщина и положение диафрагмы.


  2. PARF(1-2) = R/1-10 ,D/1-9  ;
     PARC(1-2) = R/20-11,D/19-11;
     CC(1-2)   =   -1,     1 ;

     Описаны условия связей между параметрами в гипотетической
     системе, обладающей симметрией вида :
     C(20)=-C(1),C(19)=-C(2),C(18)=-C(3),.....,C(11)=-C(10)
     D(19)=D(1),D(18)=D(2),D(17)=D(3),....D(11)=D(9);


  3. INF(1-4) = D/1-5, -D/3, D/1, D/5;
     INFT(3,4)=0.01,0.05;

     Показаны возможности корректировки списка ограничений-неравенств
     и значений нормирующих коэффициентов. В список контролируемых
     параметров включены 1, 2, 4, 5-ое осевые расстояния. Для 2-го и
     4-го осевого расстояний устанавливаются значения
     допусков, принятые по умолчанию. Для ограничения на границу
     1-го осевого расстояния значение допуска установлено = 0.01, а
     для ограничений на границу 5-го осевого расстояния - значение
     0.05


               4.2. МАРКИ КОНТРОЛИРУЕМЫХ ХАРАКТЕРИСТИК

  Марки контролируемых характеристик используется в списках:

      - оптимизируемых функций
      - ограничений-равенств
      - ограничений-неравенств

  и позволяют описать:

      - параксиальные характеристики всей системы и
        характеристики ее отдельных компонентов,
      - продольные габариты всей системы и габариты
        ее компонентов,
      - аберрации лучей осевого и внеосевых пучков,
      - коэффициенты зонального полихроматического
        разложения аберрационной функции,
      - "косые" толщины (расстояния между поверхностями
        системы ,включая поверхности предмета,изображе-
        ния и диафрагмы, вычисленные на уровне световых
        высот).


       4.2.1. МАРКИ ПАРАКСИАЛЬНЫХ И ГАБАРИТНЫХ ХАРАКТЕРИСТИК.

  Параксиальные характеристики могут относиться к оптической системе
в целом либо к ее части - блоку.

  Общий вид марки:  имя/N1*N2/пи,

где  имя - идентификатор параксиальной или габаритной характеристики;
     N1,N2 - номера первой и последней поверхностей оптического
             блока,для которого рассчитывается характеристика;
     п - тип предмета для оптического блока ( под предметом
         понимается параксиальное изображение, формируемое
         предшествующей частью оптической системы ): F - предмет
         удаленного типа, N - предмет ближнего типа, ? - предмет, по
         типу совпадающий с предметом для всей оптической системы;
     и - тип изображения для оптического блока: F - изображение
         удаленного типа, N - изображение ближнего типа,
         ? - изображение, по типу совпадающее с изображением для всей
         оптической системы.

  Если характеристика относится ко всей оптической системе в целом,
то указатели марки можно опустить. Символы ? в крайней правой части
указателей также можно не указывать.

  Параксиальные характеристики определяются  для основной длины  волны
L(0) спектрального интервала.

  Для удаленных предмета и  изображения их положение отсчитывается  от
соответствующих зрачков и измеряется  в диоптриях, а положение зрачков
отсчитывается от поверхностей системы и измеряется в мм.

  Для близких  предмета и  изображения их  положение отсчитывается  от
соответствующих поверхностей  системы и измеряется в  мм, а  положение
входного и выходного зрачков отсчитывается от поверхностей предмета  и
изображения соответственно и измеряется в диоптриях.

  Интерпретация марок: LS', LS, LSS' - зависит от типа предмета  и/или
изображения.  В  частности,  при  использовании  характеристики LS для
предмета  ближнего   типа  вычисляется   расстояние  от   предмета  до
последней  поверхности  блока,  а  для  предмета  удаленного  типа   -
расстояние от параксиального входного зрачка до последней  поверхности
блока. Аналогично интерпретируются марки LS', LSS'.

  Толщины   линз    "по    краю",   световые    высоты    и    функция
светораспределения     могут      контролироваться     только      как
ограничения-наравенства,  причем   для  AS   устанавливается    только
верхняя граница, а для ID и DE только нижняя.


               4.2.3. МАРКИ АБЕРРАЦИЙ ОТДЕЛЬНЫХ ЛУЧЕЙ

  Общий вид марки:  имя/LJIS, где

  имя - идентификатор контролируемой аберрации

         лучей и сечений. Количество указателей в марках различных
         аберраций неодинаково.

              их количества НЕДОПУСТИМО !.

  ЗАМЕЧАНИЯ:

  1. Значение  L не  должно превышать  количества дополнительных  длин
волн  в  спектральном  интервале.  Значения  длин волн и их количество
содержится в объектном модуле исходной системы и в данных  оптимизации
не корректируется.

  2. Значение  J  не  должно  превышать  количества  внеосевых пучков.
Количество  внеосевых   пучков  и   относительные  величины   предмета
содержатся в объектном модуле исходной системы и в данных
оптимизации не корректируются.

  3. Значение I не  должно превышать количества лучей,  рассчитываемых
в  соответствующих  сечениях  пучков.  По  умолчанию  в  осевом  пучке
рассчитывается 1  луч, в  половине меридионального  сечения внеосевого
пучка  -  1  луч,  в  сагиттальном  сечении  -  1  луч , косые лучи по
умолчанию не рассчитываются.

  Количество  лучей,  отличное  от  значения,  принятого по умолчанию,
может быть задано в данных оптимизации с помощью операторов вида :

      RAYA=1;   (2,3 или 4) в осевом пучке
      RAYM=1;   (2,3 или 4) в половине меридионального
                               сечения внеосевого пучка
      RAYS=1;   (2,3 или 4) в сагиттальном сечении
                               внеосевого пучка
      RAYT=1;   (2,3 или 4) в половине косого сечения
                               внеосевого пучка


  Относительные   координаты   лучей    в   сечениях   по    умолчанию
распределяются  по  закону  квадратного  корня.   При   необходимости,
относительные координаты лучей могут быть заданы операторами вида:

      PCA(*)=1,...;       в осевом пучке
      PCM(*)=1,...;       в половине меридионального
                            сечения внеосевого пучка
      PCS(*)=1,...;       в сагиттальном сечении
                            внеосевого пучка
      PCT(*)=1,...;       в половине косого сечения
                            внеосевого пучка

  Азимут  (угол  относительно  меридиональной  плоскости) для верхнего
косого сечения по  умолчанию принимается =45  градусам, но может  быть
задан оператором вида:

      SECT = 60;    ( 0 < SECT < 90).

  Нижнее  косое  сечение  внеосевого  пучка  (  вторая половина косого
сечения ) расположено симметрично верхнему относительно  сагиттального
сечения.

  Лучи  верхней  и  нижней  половины  меридионального и косого сечений
симметричны и нумеруются  от края зрачка  к центру. Симметричные  лучи
имеют одинаковые номера и различаются признаком сечения.

  4.  При  определении  первичного  и  вторичного хроматизма аберраций
внеосевых   пучков    из   меридиональных    составляющих    аберраций
дополнительных   длин   волн   предварительно   вычитаются   аберрации
хроматизма увеличения.

  5. При указателе L=' хроматические разности аберрации вычисляются
для следующих длин волн:

            L(0)      - основная длина волны
  NLA=3  -  L(2)-L(1) ─┐
  NLA=4  -  L(3)-L(2) ─┼─ дополнительные длины волн
  NLA=5  -  L(4)-L(2) ─┘

  6. При указателе L=" хроматические аберрации вторичного спектра
вычисляются для следующих длин волн:

            L(0)                - основная длина волны
  NLA=3  -  (L(2)+L(1))/2-L(0)  ─┐
  NLA=4  -  (L(3)+L(2))/2-L(0)  ─┼─ дополнительные длины волн
  NLA=5  -  (L(4)+L(2))/2-L(0)  ─┘

 7. При NLA<3 нельзя использовать символ  " (вт.спектр), а при NLA=1
-символы  '  и   " .

               4.2.3.5 МУЛЬТИПЛИКАЦИЯ МАРОК АБЕРРАЦИЙ

  Каждая марка может  идентифицировать одну аберрацию  (индивидуальная
марка) или группу однотипных  аберраций (групповая марка).   Групповая
и индивидуальная марки  имеют одинаковые имена,  но в групповой  марке
числовые указатели  хроматизма, номера  пучка, луча  или типа  сечения
заменяются символом  *.   Групповая марка  расшифровывается как  набор
марок  с   тем  же   именем,  в   которых  соответствующее    значение
индекса-указателя  "пробегает"   все  значения   от  минимального   до
максимального с шагом  1, (например, номер  пучка может меняться  от 1
до  NPEN,  где  NPEN  -  кол-во внеосевых пучков), или последовательно
заменяется элементами стандартного набора чисел и символов  (например,
для указателя типа  сечения - символами  'U'-верхнее и 'D'  -нижнее) .
Вместе  с   маркой  мультиплицируются   и  параметры   соответствующих
функций, имеющие  тот же  индекс в  массиве значений,  что и групповая
марка (весовые коэффициенты, нормы и заданные значения  характеристик,
если  речь  идет  об  оптимизируемых  функциях,  и граничные значения,
нормы   и   допуски,    если   марка    является   элементом    списка
ограничений-неравенств)

  ПРИМЕР 14. Групповая марка продольных аберраций осевого пучка имеет
             вид: DSA/**. Пусть в спектральном диапазоне задано 3
             длины волны, а в осевом пучке рассчитывается 2 луча.

             Тогда описание:

             MF(1) = DSA/**;
             MFW(1) = 1;
             MF#(1) = 0;

             эквивалентно описанию :

             MF(1-8) = DSA/01, DSA/11, DSA/21, DSA/02,
                       DSA/12, DSA/22, DSA/10, DSA/20;
             MFW(1-8) = 1,1,1,1,1,1,1,1;
             MF#(1-8) = 0,0,0,0,0,0,0,0;

  В  списке  можно  указать  несколько  марок  аберраций  одного  типа
(индивидуальных или групповых)  с разными параметрами  соответствующих
функций.  Каждая  последующая  марка  и  соответствующие  ей параметры
функций корректируют набор  марок и параметров,  описанный предыдущими
марками. Если марка характеристики еще не встречалась в наборе, то  он
дополняется текущей маркой с соответствующими параметрами. Если  марка
есть в  наборе, то  значения соответствующих  параметров заменяются на
значения, связанные  с текущей  маркой, причем  по умолчанию  значения
параметров  сохраняются   прежними.  Если   текущая  марка
является групповой, то она мультиплицируется (см.пример 14), и
полученный  набор марок  и  параметров  рассматривается  как
корректирующий для набора, сформированного  предыдущими  марками.
Марка  характеристики   может записываться со знаком  '-' (минус)
перед  ее символическим именем.  В этом   случае   такая   же
марка   или   группа   марок  вместе  с соответствующими
параметрами   удаляются   из   набора,   описанного предыдущими
марками.

  ПРИМЕР 15. при условиях примера 14 описание

             MF(1-4)=DSA/**,-DSA/2*,DSA/1*,DSA/01;
             MFW(3,4)=0.5,2;
             MF#(1-3)=0; MF#(4)=0.01;

             эквивалентно описанию

             MF(1-5)=DSA/01,DSA/11,DSA/02,DSA/12,DSA/10;
             MFW(1-5)= 2,    0.5,    1,     0.5,   0.5 ;
             MF#(1-5)=0.01,  0,      0,      0,     0 ;

      4.2.5. МАРКИ,ОПИСЫВАЮЩИЕ ОТСТУПЛЕНИЕ ЧАСТОТНО-КОНТРАСТНОЙ
             ХАРАКТЕРИСТИКИ (ЧКХ) ОТ БЕЗАБЕРРАЦИОННОЙ.

  ПРИМЕЧАНИЯ.  1. Марки, описанные в настоящем разделе, можно
                  использовать при поставке специального программного
                  обеспечения по дополнительному заказу.

               2. Марки используются только в работе OPTW
                  преимущественно для дифракционно-ограниченных
                  систем.

               3. При использовании данных марок для повышения
                  эффективности оптимизации настоятельно
                  рекомендуется использование "виртуального" диска,
                  для чего в файле AUTOEXEC.BAT должна присутствовать
                  строка:

                  SET OPAL_TEMP=G:\

                  где G - это "виртуальный" диск.



  Отступления описываются маркой:

      MTF/IJK

где   I - номер пучка ( 0 < I < NPEN )
      J - номер частоты  ( J=1,2,3,4 )
      K - номер сечения ( 1 - сагиттальное, 2 - меридиональное )

  Для  полного  описания необходимо  указать  количество  и  значения
частот, на которых необходимо произвести коррекцию отступления от ЧКХ.

 - Количество частот описывается маркой:

      NFRQ = к;

где   к - количество частот   ( 1 <= k <= 4 )

 - Значения частот описываются маркой:

      FRQ(1-к)= ч1,ч2,...чк;

где   к - количество частот
      ч1,ч2,...чк - корректируемые частоты


  ПРИМЕР 16. Пусть необходимо минимизировать отступления от
             безаберрационной ЧКХ для осевого и 1-ого пучков в
             сагиттальном и меридиональном сечениях на частотах 50 и
             100 лин/мм. Это можно описать следующим образом:

             MF(*) = MTF/011, MTF/111,MTF/112,
                     MTF/021, MTF/121,MTF/122;
             NFRQ = 2;
             FRQ(1-2) = 50, 100;




              5.  СЕАНС  РАБОТЫ  С  ОПТИМИЗАТОРОМ


  Сеанс  работы  с  оптимизатором  начинается  с  вызова  в меню работ
работы  "ОПТИМИЗАЦИЯ   ГЕОМЕТРИЧЕСКИХ  АБЕРРАЦИЙ"   или   "ОПТИМИЗАЦИЯ
ВОЛНОВЫХ  ФРОНТОВ"  или  набора  в  командной строке команды "OPT" или
"OPTW" соответственно.

  После вызова соответствующего модуля оптимизации появляется  главное
меню.  Для  начала  работы  необходимо  произвести  трансляцию  данных
оптимизации,  выбрав  1-й  пункт  меню.  При  этом будет запрошено имя
файла  с  данными.  По  умолчанию  подставляется  имя файла оптической
системы.  Если  файл  с  введенным   именем  не  существует,  то   для
трансляции  подставляется  шаблон,  который  можно  модифицировать   с
учетом оптической  системы. Если  файл существует,  то для  трансляции
загрузятся  соответствующие  данные.  Трансляция данных осуществляется
нажатием  клавиши  "F4".  Если  в  тексте  имеются ошибки, то появятся
соответствующие  сообщения.   Ошибки  необходимо  исправить  и   снова
произвести трансляцию. При успешной трансляции сообщений не  выдается.
После  того,  как  задание  на  оптимизацию  будет  отредактировано  и
странслировано,  его  можно  сохранить,  нажав  функциональную клавишу
"F8". Выход из  транслятора производится с  помощью клавиши "ESC"  или
"F10". При выходе автоматически осуществляются операции по  подготовке
к оптимизации, мультипликация марок, расчет и аппроксимация  аберраций
в исходной  точке, вычисление  штрафов и  формирование выходных данных
для просмотра. При успешном  завершении выше перечисленных действий  и
при условии, что режимы оптимизации, краткая сводка которых  приведена
во второй  информационной строке  сверху, удовлетворяют  требованиям к
системе, можно перейти непосредственно к процессу оптимизации,  выбрав
пункт  3  меню.  Если  режимы  оптимизации необходимо изменить, то это
можно сделать, выбрав пункт 2 меню.

  При выборе пункта  3 меню на  экране появятся таблицы,  отображающие
ход оптимизации  по циклам,  шагам и  основным частям  шага.  Просмотр
хода  возможен  как  в  текстовом,  так  и  в графическом режимах.  По
умолчанию   устанавливается   текстовый   режим.    Для   перехода   в
графический  необходимо  нажать  клавишу  "F6",  после  чего на экране
будут отображены:

 - графики изменения критерия качества (зеленым цветом) и изменения
штрафа (красным цветом) по шагам и циклам оптимизации;

 - диаграмма распределения вкладов оптимизируемых функций в критерий
   качества коррекции. Серым цветом показано распределение
   оптимизируемых функций на начало оптимизации, бежевым -
   распределение оптимизируемых функций на текущем шаге.  В строке
   "МАСШТАБ" соответствующими цветами показано соотношение максимальных
   вкладов на текущее состояние и на начало оптимизации.

 - диаграмма распределения вкладов ограничений-неравенств в штрафную
   функцию, если она не нулевая.
 
 - отображение  оптической схемы.

Переход обратно в текстовый  режим производится клавишей "F6".

  При   необходимости   завершить   оптимизацию,   следует   во  время
выполнения анализа  влияния параметров  нажать "ESC"  или "F10", после
чего появится  меню, которое  будет содержать  запрос о  подтверждении
окончания процесса.  Если клавиша  "ESC" была  нажата случайно,  то от
завершения можно отказаться, выбрав соответствующий пункт.

      - Обобщенное увеличение в заданной плоскости  V0,

                            V0/*
                               │
         номер состояния       │
         ──────────────────────┘


      - Обобщенное увеличение в плоскости Гаусса  VG0,

                             VG0/n1-n2/FN0
                                 │   │ │││   номер состояния
    номер начальной поверхности  │   │ ││└──────────────────────────
    ─────────────────────────────┘   │ ││
                                     │ ││                  ┌─ F - дальний
    номер конечной поверхности       │ ││ тип изображения ─┴─ N - ближний
    ─────────────────────────────────┘ │└───────────────────────────
                                       │
                                       │                ┌─ F - дальний
                                       │  тип предмета ─┴─ N - ближний
                                       └────────────────────────────




          Положение плоскости предмета (для блока) ......  SG

                                 SG/n1/F0
                                    │  ││   номер состояния
    номер начальной поверхности     │  │└──────────────────────────
    ────────────────────────────────┘  │
                                       │                ┌─ F - дальний
                                       │  тип предмета ─┴─ N - ближний
                                       └────────────────────────────


          Положение плоскости Гаусса ....................  SG'

                                SG'/n2/F0
                                    │  ││   номер состояния
    номер конечной поверхности      │  │└──────────────────────────
    ────────────────────────────────┘  │
                                       │                   ┌─ F - дальний
                                       │  тип изображения ─┴─ N - ближний
                                       └────────────────────────────



    Смещение плоскости изображения относительно 0-состояния  DS'0

                          DS'0/*
                               │
         номер состояния       │
         ──────────────────────┘

                                     DSA/*"*  - вторич. хром.

                     номер состояния
