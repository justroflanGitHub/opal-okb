# Руководство пользователя OPAL-OKB

## Установка

```bash
pip install PyQt5 numpy
py main.py
```

Требуется **Python 3.10+** (рекомендуется 3.13). На Windows используется `py` launcher.

---

## Интерфейс

1. **Меню** — Файл, Система, Вид
2. **Таблица поверхностей** — параметры элементов
3. **Параметры системы** — предмет, апертура, λ, точки поля
4. **Визуализация** — ход лучей
5. **Панель анализа** — 13 вкладок графиков
6. **Строка состояния** — f', BFD, суммы Зейделя

---

## Таблица поверхностей

| Столбец | Описание |
|---------|----------|
| № | Номер поверхности |
| R (мм) | Радиус кривизны, 0 = ∞ (плоскость) |
| d (мм) | Расстояние до следующей поверхности |
| Стекло | Марка или пусто (воздух) |
| D/2 (мм) | Полудиаметр |
| k | Коническая постоянная (0 = сфера) |
| Тип | SPHERE / CONIC / ASPHERIC |

### Асферические поверхности

Столбец **k** задаёт коническую постоянную:
- `k = 0` — сфера
- `k = −1` — параболоид
- `−1 < k < 0` — эллипсоид
- `k < −1` — гиперболоид

Для полиномиальных асферик (A4–A10) используйте тип ASPHERIC. Коэффициенты задают отклонение от сферы.

### Стоп-поверхность

Чекбокс «Стоп» в строке поверхности устанавливает апертурную диафрагму. Стоп-поверхность подсвечивается голубым.

## Параметры системы

### Предмет
- **Бесконечность** — поле в градусах
- **Конечное** — поле в мм

### Апертура
- Входной зрачок (мм) / NA / F/#

### Длины волн
До 5 длин волн (мкм). Стандартные: d=0.58756, F=0.48613, C=0.65627.

### Точки поля
Виджет «Точки поля» — до 5 внеосевых пучков. Для каждой точки:
- Y-координата (град/мм)
- Вес
- **Виньетирование** — верхний и нижний пределы зрачка (0…1)

Виньетирование позволяет ограничить зрачок для каждого поля отдельно (отсечь затенённые лучи).

### Полихроматический режим
Если задано несколько длин волн, анализ автоматически выполняется для всех λ с весами. Графики показывают кривые для каждой λ, а также полихроматическую сумму.

## Анализ — 13 вкладок

### 1. Точечная диаграмма
Пятно рассеяния, RMS диаметр.

### 2. Δy' — Поперечная аберрация
Δy' от зрачковой координаты.

### 3. Δs' — Продольная аберрация
Продольное смещение фокуса от зрачковой координаты.

### 4. W — Волновая аберрация
Отклонение волнового фронта в λ. Хорошо скорректированная: |W| < λ/4.

### 5. ЧКХ геометрическая
Геом. MTF через FFT.

### 6. ЧКХ дифракционная
Дифр. MTF через FFT. Предельная частота: ν₀ = 1/(λ·F/#).

### 7. Дисторсия
Относительная дисторсия по полю.

### 8. Астигматизм + кома
Тангенциальный / сагиттальный фокус по полю.

### 9. Фокусировка
RMS пятна vs defocus. Находит оптимальную плоскость изображения.

### 10. PSF — Point Spread Function
Дифракционная функция рассеяния точки. 2D карта интенсивности в фокальной плоскости. Показывает реальное распределение энергии в изображении точечного источника.

### 11. LSF — Line Spread Function
Интеграл PSF по одному направлению. Характеризует отклик на щелевую миру.

### 12. ENC — Encircled Energy
Доля энергии внутри круга заданного радиуса. Показывает, какая часть света попадает в пиксель/диафрагму.

### 13. PTF — Phase Transfer Function
Фазовая часть оптической передаточной функции. Вносит сдвиг в пространственные гармоники изображения.

## Система

### Обернуть
Разворот системы (первый ↔ последний).

### Масштаб
Умножение всех R и d на коэффициент.

### Стандартные радиусы
Замена R на ближайшие по ГОСТ 1807-75 (ряд R40).

### Ахромат
Авто-расчёт ахроматического дублета. Параметры: f', крон, флинт, апертура. Условие: φ₁/ν₁ + φ₂/ν₂ = 0.

### Подгонка
Подгонка характеристик:
- **f'** — подбор радиуса для заданного фокусного расстояния
- **BFD** — подбор радиуса для заданного заднего отрезка
- **Увеличение** — подбор радиуса для заданного увеличения

Метод: binary search по радиусу указанной поверхности.

## Устранение неполадок

**«ModuleNotFoundError: No module named 'PyQt5'»**
```bash
pip install PyQt5
```

**Каталог стёкол не найден (889 марок)**
Полный каталог загружается из `extracted/opal_okb/*.FIL`. Убедитесь, что директория содержит GCTG.FIL, FCTG.FIL, HCTG.FIL, GCNG.FIL.

**Ошибка импорта .OPJ**
Не все .OPJ файлы содержат валидные системы. 83 файла парсятся, 63 с именами стёкол.
